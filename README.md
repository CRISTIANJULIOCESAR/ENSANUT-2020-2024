# Armonización de Módulos ENSANUT Continua (2020–2024)

Este repositorio documenta el trabajo de armonización, limpieza e integración de módulos de la ENSANUT Continua para los años 2020, 2021, 2022, 2023 y 2024.

## Descripción

El trabajo realizado consistió en la **armonización de los módulos de cada levantamiento a nivel anual**, permitiendo construir bases consolidadas por año a partir de la integración de componentes socioeconómicos, de salud, nutrición y antropometría.

## Alcance

Se logró:

- armonizar variables y catálogos dentro de cada año;
- limpiar y validar registros;
- integrar módulos temáticos por levantamiento;
- generar bases anuales consolidadas para análisis.

## Limitación principal

No fue posible integrar todos los años en una sola base histórica interoperable, debido a diferencias en estructura, variables, catálogos y criterios metodológicos entre levantamientos.

## Módulos trabajados

- Socioeconómico
- Salud
- Nutrición
- Antropometría
